# Power Platform

## Power Apps
A lightweight app to create and edit entries in the SharePoint list `HousingEvents`.

![Power Apps](power-platform/power-apps.png)

## Power Automate
Flow triggered when a new item is created in `HousingEvents`, sending an email notification.

![Power Automate](power-platform/power-automate.png)

## SharePoint
`HousingEvents` list stores the curated event log used by Power BI.

![SharePoint](power-platform/sharepoint-list.png)
